# Quokkawai Project Overview

## 1. 프로젝트 개요

Quokkawai는 사용자의 스마트폰 앱 사용 시간을 기록하고, 하루 단위로 사용 패턴을 분석하여 디지털 디톡스를 돕는 서비스이다.

사용자는 자신의 하루 스마트폰 사용량, 가장 많이 사용한 앱, 앱 카테고리별 사용 시간을 확인할 수 있다. 또한 AI가 사용 패턴을 바탕으로 간단한 피드백과 다음 행동 추천을 제공한다.

---

## 2. 문제 정의

스마트폰 사용자는 자신이 어떤 앱에 시간을 많이 쓰는지 정확히 인지하지 못하는 경우가 많다.

특히 SNS, 영상, 게임 앱은 짧은 시간만 사용한다고 느껴도 실제 누적 사용 시간이 길어질 수 있다. 기존의 단순 사용 시간 표시만으로는 사용자가 자신의 행동 패턴을 이해하고 조절하기 어렵다.

Quokkawai는 앱 사용 데이터를 수집하고, 이를 카테고리별로 정리한 뒤, AI 피드백을 제공하여 사용자가 스스로 사용 습관을 점검할 수 있도록 한다.

---

## 3. 서비스 목적

Quokkawai의 목적은 사용자의 스마트폰 사용을 강제로 차단하는 것이 아니다.

서비스의 핵심 목적은 다음과 같다.

```text
1. 사용자의 앱 사용 시간을 기록한다.
2. 하루 단위로 사용량을 요약한다.
3. 많이 사용한 앱과 카테고리를 보여준다.
4. AI가 사용 패턴에 대한 짧은 피드백을 제공한다.
5. 사용자가 자신의 스마트폰 사용 습관을 인식하도록 돕는다.
```

---

## 4. 핵심 기능

### 4.1 Google 로그인

사용자는 Google 계정으로 로그인한다.

초기 개발 단계에서는 더미 userId를 사용할 수 있지만, 최종 구현에서는 Google OAuth 기반 인증을 사용한다.

---

### 4.2 앱 사용량 수집

Flutter 앱은 Android 기기에서 앱별 사용량 데이터를 수집한다.

MVP에서는 세션 단위 로그가 아니라 하루 앱별 요약 로그를 서버에 전송한다.

예시:

```json
{
  "date": "2026-05-01",
  "packageName": "com.google.android.youtube",
  "appName": "YouTube",
  "usageSeconds": 5400,
  "openCount": 12
}
```

---

### 4.3 사용 로그 저장

FastAPI 서버는 Flutter 앱에서 전달받은 사용 로그를 Firestore에 저장한다.

사용 로그는 사용자, 날짜, 앱 패키지명 기준으로 관리한다.

---

### 4.4 일일 사용 요약

서버는 특정 날짜의 사용 로그를 기반으로 다음 정보를 계산한다.

```text
총 사용 시간
상위 사용 앱 3개
카테고리별 사용 시간
```

---

### 4.5 AI 사용 분석

서버는 일일 사용 요약 데이터를 OpenAI API에 전달하고, 사용자의 하루 사용 패턴에 대한 짧은 피드백을 생성한다.

AI는 원본 로그 전체를 직접 분석하지 않는다. 서버가 먼저 요약 데이터를 만든 뒤, 요약된 정보만 AI에 전달한다.

## 6. MVP 범위

MVP에서 구현할 기능은 다음과 같다.

```text
1. Google 로그인
2. 앱 사용량 수집
3. 사용 로그 업로드
4. Firestore 저장
5. 일일 사용 요약 조회
6. 카테고리별 사용량 조회
7. AI 한줄 피드백 생성
```

---

## 7. MVP 제외 범위

다음 기능은 MVP 범위에서 제외한다.

```text
실시간 앱 차단
앱 사용 강제 제한
친구 랭킹
주간/월간 통계
화면 내용 분석
세션 단위 정밀 추적
사용자별 맞춤 AI 장기 기억
```

---

## 8. 최종 서비스 흐름

```text
1. 사용자가 Google 계정으로 로그인한다.
2. Flutter 앱이 Android 앱 사용량을 수집한다.
3. 앱이 하루 앱별 사용량 요약을 서버에 전송한다.
4. FastAPI 서버가 사용 로그를 Firestore에 저장한다.
5. 서버가 특정 날짜의 일일 요약을 계산한다.
6. 서버가 요약 데이터를 기반으로 AI 피드백을 생성한다.
7. Flutter 앱이 사용량 리포트와 AI 피드백을 화면에 표시한다.
```

---

## 9. 프로젝트 한 줄 정의

Quokkawai는 Flutter 앱에서 수집한 하루 앱별 사용량 요약 데이터를 FastAPI 서버로 전송하고, Firestore에 저장한 뒤, 일간 사용 요약과 AI 피드백을 제공하는 디지털 디톡스 서비스이다.
