# Quokkawai 프로젝트 개요

## 1. 프로젝트 한 줄 정의

스마트폰 사용 데이터를 수집하고 AI로 사용 맥락을 분석하여, 사용자의 디지털 과몰입 패턴을 시각화하고 개선 피드백을 제공하는 디지털 디톡스 앱

---

## 2. 문제 정의

현대 사용자는 스마트폰 사용 시간이 많아지고 있지만, 자신이 어떤 앱을 어떤 목적으로 얼마나 사용하는지 정확히 인식하지 못하는 경우가 많다.

단순 사용시간 제한 앱은 사용자의 실제 사용 맥락을 반영하지 못한다. 예를 들어 YouTube를 사용하더라도 강의 시청일 수도 있고, 쇼츠 소비일 수도 있다.

따라서 앱 사용시간뿐만 아니라 사용 맥락을 함께 분석하여 사용자에게 더 의미 있는 피드백을 제공하는 서비스가 필요하다.

---

## 3. 핵심 목표

- 앱 사용시간 수집
- 앱별 사용량 통계 제공
- 사용 목적 기반 카테고리 분류
- AI 기반 하루 사용 패턴 피드백 제공
- 사용자가 자신의 디지털 습관을 인식하도록 도움

---

## 4. MVP 기능

| 기능 | 설명 |
|---|---|
| 앱 사용 로그 수집 | 앱 이름, 패키지명, 사용시간 수집 |
| 앱별 사용시간 통계 | 하루 기준 앱별 사용량 계산 |
| TOP3 앱 표시 | 많이 사용한 앱 3개 표시 |
| AI 카테고리 분류 | 공부, 생산성, SNS, 게임 등으로 분류 |
| AI 한줄 피드백 | 하루 사용 패턴에 대한 짧은 피드백 |
| 일간 리포트 | 총 사용시간, TOP 앱, 피드백 표시 |

---
